Privacy Policy

A short overview of what this portfolio does and doesn't do with data.

1. Overview

This site is a personal portfolio built to showcase my work as a full stack developer. It does not collect analytics, does not track visitors, and does not use cookies for profiling.

2. What data is processed

When you visit this site, standard technical information is processed by the hosting provider (Netlify), such as IP address, browser type, and basic request logs. These logs are used only for security and performance and are not used by me to profile or identify individual visitors.

The projects page reads public data from my GitHub profile using the GitHub REST API. This only fetches information about my own repositories; it does not read anything about you.

3. Contact

If you contact me by email, LinkedIn, or phone, any personal data you share in that message stays in those channels. I do not copy it into separate databases or sell it to anyone.

4. Your rights

If you have any questions about data related to this site, you can contact me at sage.stockmans@pm.me. I'll respond as soon as reasonably possible.
